# glossario-ia.md — Glossário de IA para classificação correta

> Termos que o agente precisa reconhecer e classificar corretamente ao pesquisar tendências.

## Modelos e arquiteturas
| Termo | O que é | Categoria no painel |
|---|---|---|
| LLM | Large Language Model — modelo de linguagem grande | Novos lançamentos de modelos |
| Multimodal | Modelo que processa texto + imagem (e/ou áudio, vídeo) | Novos lançamentos de modelos |
| Context window | Tamanho máximo de texto que o modelo processa de uma vez | Novos lançamentos de modelos |
| Fine-tuning | Ajuste fino de um modelo em dados específicos | IA para desenvolvedores |
| Benchmark | Teste padronizado de desempenho — sozinho não é tendência | (filtrar; só incluir com contexto prático) |
| Open-source / open-weight | Modelo com pesos disponíveis publicamente | Novos lançamentos de modelos |

## Infraestrutura e dev
| Termo | O que é | Categoria no painel |
|---|---|---|
| MCP | Model Context Protocol — padrão de integração de ferramentas com LLMs (Anthropic) | IA para desenvolvedores |
| API | Interface de programação — acesso programático a modelos | IA para desenvolvedores |
| RAG | Retrieval-Augmented Generation — combina busca com geração | IA para desenvolvedores |
| Embedding | Representação vetorial de texto para busca semântica | IA para desenvolvedores |
| Inference | Execução do modelo pra gerar uma resposta | IA para desenvolvedores |
| Token | Unidade de processamento do LLM (~¾ de palavra em inglês) | (contexto de preço e custo) |

## Agentes e automação
| Termo | O que é | Categoria no painel |
|---|---|---|
| Agente / Agent | Sistema que usa LLM para planejar e executar tarefas autonomamente | Agentes de IA e automações |
| Orquestrador | Sistema que coordena múltiplos agentes | Agentes de IA e automações |
| Tool use / Function calling | Capacidade do modelo de chamar ferramentas externas | Agentes de IA e automações |
| n8n / Make | Plataformas de automação no-code/low-code | IA para empreendedores |
| Workflow | Fluxo de trabalho automatizado | Agentes de IA e automações |

## Termos de mercado
| Termo | O que é |
|---|---|
| Valuation | Avaliação de mercado de uma empresa (não é tendência sozinho) |
| Series A/B/C | Rodadas de investimento (relevante se impactar produto disponível) |
