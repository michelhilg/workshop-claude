# Workshop Claude — IA Sem Frescura

Compilado dos materiais apresentados no workshop de uso prático do Claude Code. O repositório reúne exemplos, cases e slides utilizados durante o evento, organizados para facilitar o estudo e a referência posterior.

---

## Estrutura do Repositório

### `01_comece_aqui/`
Ponto de partida para quem está chegando ao workshop. Contém os materiais introdutórios e os primeiros exercícios para se familiarizar com o Claude Code.

### `02_cases_completos/`
Cases práticos completos, do início ao fim. Cada subpasta representa um caso de uso real trabalhado durante o workshop, com todos os arquivos necessários para reproduzir o exemplo.

### `03_apresentacao/`
Slides e materiais de apoio utilizados na apresentação do workshop.

---

## Sobre o Workshop

O workshop faz parte da série **IA Sem Frescura** — uma iniciativa focada em mostrar aplicações práticas de IA no dia a dia, sem enrolação.
